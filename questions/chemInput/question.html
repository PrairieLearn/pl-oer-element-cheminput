Questions below were designed to showcase the different features of pl-chem-input The goal of pl-chem-input is to allow users to enter chemical equation. The correct answer in all questions below is 2H2O(l) -> 2H2(g) + O2(g).

The states of matter will not be graded unless stated otherwise. Including them when they are not graded will not cause you to lose points, but you will be penalized for excluding them when they are graded.

Feedback will be provided unless stated otherwise.

Part 1

The following demonstrates the default behavior of the pl-chem-input element. This example illustrates a question with a weight of 1 and a placeholder text set to "Your answer here." The allow-blank attribute is set to false, requiring students to provide an answer before submission. Additionally, the help text is displayed, and the editor's default size is set to 100, spanning the entire line. The score badge will appear once the "Submit Answer" button is clicked.

Electrolysis Reaction: Instruction: Write the chemical equation for the electrolysis of water (H2O) into hydrogen gas (H2) and oxygen gas (O2).

Part 2

This question demonstrates the use of different sizes for the pl-chem-input element on the page. Specifically, the input field in this example is configured with the size="50" attribute, resulting in a width of 50%. Additionally, the weight assigned to this question is higher compared to the preceding ones with configuration of weight="5". Lastly, the grade-states="true"attribute will ensure that the states of matter are graded as well.
